What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects and allows programming languages to interact with the documentâ€™s content, structure, and styles.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that allows you to attach a single event listener to a parent element to manage events for multiple child elements. This is useful for performance and maintaining clean code, especially when elements are added or removed dynamically.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute() and element.getAttribute(). To change styles, use the element.style property.
